Overview

Angular Trianglify Animate is tiny (2kb) plugin to add support for animate images SVG to your page.

Default options

ng-speed = "1" // Animation speed  - number	
ng-easing = "linear" // avaiable: 'in-out-cubic', 'in-out-quintic', 'in-quintic', 'in-quartic', 'in-cubic', 'in-quadratic', 'out-quintic', 'out-quartic', 'out-cubic', 'out-in-cubic', 'out-in-quartic', 'back-in-cubic', 'back-in-quartic', 'out-back-cubic', 'out-back-quartic', 'out-elastic-small', 'out-elastic-big'

Experiments options

ng-vector-x = ""
ng-vector-y = ""
ng-base-max = ""
ng-base-min = ""
ng-theta = ""
